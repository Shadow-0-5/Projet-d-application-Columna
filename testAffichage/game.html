Règles du jeu

But du jeu : Avoir le plus de pions sur les piles les plus hautes en fin de partie.

Chaque tour, le joueur doit effectuer 2 actions obligatoires dans cet ordre :

  1. Déplacer un pion : Un des quatre pions doit être déplacé vers la première pile vide en face dans une direction orthogonale (pas de case occupée par un autre pion, pas de case vide).
  2. Empiler des dalles : Choisir une pile sans pion, l'empiler sur la première pile sans pion rencontrée dans une direction orthogonale. Maximum 5 dalles par pile. La case de départ devient vide.

Fin : Quand un joueur ne peut plus jouer, c'est à dire qu'une des deux actions obligatoire ne peut pas être effectuée. Le vainqueur est celui avec le plus de pions sur les piles de 5 dalles, puis 4, puis 3, etc.
